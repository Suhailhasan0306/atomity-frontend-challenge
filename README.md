# Atomity Frontend Engineering Challenge

This project is my submission for the **Atomity Frontend Engineering Challenge**.  
The goal of this project was to recreate a **scroll-triggered animated metrics section** inspired by the product video.

The section displays important cloud infrastructure metrics such as CPU usage, active instances, cost savings, and system efficiency.  
Each metric is presented in an animated card with smooth entrance animations and hover interactions to create a clean and modern user experience.

---

## Features

- Scroll-triggered animations using **Framer Motion**
- Staggered card animations for better visual flow
- Hover interactions with elevation and glow effect
- Animated metric counters for dynamic data display
- Data fetched from a **public API**
- Efficient data caching using **React Query**

---

## Tech Stack

- **React (Vite)** – for building the UI
- **Framer Motion** – for smooth animations
- **TanStack React Query** – for data fetching and caching
- **Modern CSS** – responsive grid layout and styling

---

## Component Structure

src/
components/
FeatureSection.jsx // main animated section
AnimatedCard.jsx // reusable metric card component
AnimatedCounter.jsx // animated number counter
hooks/
useApiData.js // API fetching with React Query
tokens/
tokens.js // design tokens (colors, styles)


This structure keeps the project modular and makes components reusable and easier to maintain.
